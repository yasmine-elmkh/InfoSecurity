# InfoSecurity
Projet basé sur la version attribuée le 05/05/2025 de "Cybersecurity_VictorElias", réalisé dans le cadre d’un projet académique de sécurité des systèmes. Objectif : sécurisation complète d’un site web à l’aide de techniques modernes de cybersécurité (authentification, chiffrement, contrôle d’accès, journalisation, etc.).
