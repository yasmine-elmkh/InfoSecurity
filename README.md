--------------------------------------------InfoSecurity--------------------------------------------------

Ce projet est une reprise de la version du dépôt "Cybersecurity_VictorElias" attribuée le 5 mai 2025, dans le cadre d’un travail 
pratique en sécurité des systèmes. L’objectif principal est d’implémenter une solution complète de sécurisation d’un site web existant,
en y intégrant notamment : une authentification sécurisée par JWT, des rôles d’accès, un contrôle des permissions,
la journalisation des activités, ainsi que des protections contre les vulnérabilités courantes (injections, XSS, CSRF, etc.).

Voici le lien du repo originale du projet: https://github.com/250428-PracticeProjects/Cybersecurity_VictorElias
